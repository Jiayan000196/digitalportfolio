My Work

Website Redesign and Rebranding Project : SkyMap Aviation

×

My Contribution and Project Outcome

I collaborated with three team members on the rebranding and redesign of SkyMap Aviation's website. I contributed by creating the project charter, personas, moodboard, sitemap, wireframes, and completing the final redesign. The outcome was a modern, user-friendly website that improved user experience and better reflected the brand's identity.

Website Mockup Project : Frontend Design

×

My Contribution and Project Outcome

I developed a responsive news website homepage using HTML, CSS, and JavaScript. I implemented media queries for mobile-first responsiveness and applied hover and focus states to enhance interactivity and accessibility. Flexbox and CSS Grid were used to structure the layout across different screen sizes. The outcome is a pixel-perfect, fully responsive homepage that delivers a seamless user experience on all devices.

UX/UI Interactive app design project : CareFinder

×

My Contribution and Project Outcome

As a team, we designed CareFinder, a mobile app that tracks your loved ones in real time for enhanced safety and peace of mind. I contributed to the initial brainstorming, created lo-fi and hi-fi wireframes in Figma, and conducted usability testing using Maze. Based on user feedback, I refined the design and transitioned it into high-fidelity prototypes for both phone and tablet. The final outcome was a visually polished, user-friendly app prototype ready for potential commercialization.

Capstone Project : Interactive Quiz Website

×

My Contribution and Project Outcome

I designed and developed a responsive and dynamic quiz web application from scratch, applying skills from Web Development I, II, and III. I began by creating detailed mockups that included layout, UI elements, and branding across multiple screen sizes. Then built a static prototype using HTML and CSS, followed by a fully functional version using HTML, CSS, JavaScript, and Bootstrap. The final web app features dynamic interactions. The project demonstrates my ability to manage a complete development workflow—from design to deployment—using modern front-end technologies.
